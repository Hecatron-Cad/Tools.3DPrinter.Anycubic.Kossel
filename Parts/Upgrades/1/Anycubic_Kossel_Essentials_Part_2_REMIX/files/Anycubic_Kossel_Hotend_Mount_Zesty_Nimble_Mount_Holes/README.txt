# Summary

Hotend mount with mounts for extruder fan, cooling fan and Zesty Nimble.

One mounting help on the side to fix the cables.

Matches E3Dv5 hotend on Anycubic Kossel effector base.

To complete the mount for the Nimble you need this thing: https://www.thingiverse.com/thing:2369799
Use the 4 mm PTFE variant, since this holder also has a 4 mm hole.

Hotend fan: 40x40x10, axial, 32 mm square screw spacing, M3.
Cooling fan: 40x40x10, radial, 35 mm square screw spacing, M2.
Mount on effector: M3.
Mount of Nimble: M3.

I added some stabilisation to the latest design. It is not shown in the picture.

# Print Settings

Rafts: No
Supports: No
Resolution: 0.4 mm
Infill: 20 %

Notes: 
Layer Height: 
- 0.2 mm, ok
- 0.1 mm, great

# Post-Printing

You need to cut the M2 and M3 threads for the fans, the effector mount and the Nimble with an M2 and M3 thread cutter. The core holes are 1.6 mm and 2.5 mm.

You need to use one M3 nut for one Zesty Nimble mount screw.
